HTML Basics

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design the structure of web pages. It provides the foundation of every website by organizing content into headings, paragraphs, images, links, forms, and more. Web browsers read HTML files and display them visually to users.

Why Do We Use HTML?
  • To structure content on the web.
  • To define elements like text, images, buttons, and links.
  • To build forms for user input.
  • To create tables, lists, and layouts.
  • To act as the skeleton of a web page, while CSS adds style and JavaScript adds interactivity.
  • Without HTML, web pages would have no structure and nothing to display in browsers.
What Are Paired and Unpaired Tags in HTML?

In HTML, tags are used to define elements. These tags can be categorized into two types:

Paired Tags (Container Tags)

Paired tags come in two parts:

  1. Opening tag starts the element (e.g., <p>)
  2. Closing tag ends the element (e.g., </p>)

Everything between the opening and closing tag is the content of the element.

Example: <p>This is a paragraph.</p>

Unpaired (Self-closing) Tags

These tags do not have closing tags. They are used alone to insert elements like line breaks, images, or horizontal lines.

Examples:

  • <br> Line break
  • <hr> Horizontal line
  • <img src="image.jpg" alt="Image"> Image
  • <input type="text"> Input field
